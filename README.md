# Task-Management-System-with-Mongo-DB
A full-stack Task Management System developed using HTML, CSS, JavaScript, Node.js, and MongoDB. The application allows users to create, manage, update, and organize daily tasks with real-time database storage and retrieval.

Features:
Add and Manage Tasks
Update and Delete Tasks
Real-Time Data Storage with MongoDB
Responsive User Interface
Backend Integration using Node.js

Technologies Used:
HTML
CSS
JavaScript
Node.js
Express.js
MongoDB

Project Workflow:
User creates a task/reminder.
Task data is sent to the backend server.
MongoDB stores the task information.
Users can update, view, or delete tasks dynamically.

Learning Outcomes:
Full-Stack Application Development
CRUD Operations
MongoDB Database Integration
REST API Basics
Backend and Frontend Connectivity

Future Enhancements:
User Authentication System
Task Deadline Notifications
Priority-Based Task Management
Search and Filter Features
Cloud Database Deployment
